LESSONS LEARNED & BEST PRACTICES
================================================================

REACT DEVELOPMENT LESSONS:
--------------------------
1. Always destructure useState correctly: [state, setState]
2. Give unique names to functions and state variables
3. Define proper TypeScript interfaces for component props
4. Pass all required props from parent components
5. Declare all refs used in JSX
6. React hooks must be called at top level of components

FILE HANDLING LESSONS:
----------------------
1. Use FileReader for converting files to base64
2. Always validate file types and sizes before processing
3. Handle FileReader errors appropriately
4. Use data URLs for consistent image data format

UI/UX LESSONS:
--------------
1. Provide clear visual feedback for interactive elements
2. Make drag targets large and prominent
3. Use consistent styling patterns
4. Center content for better visual hierarchy
